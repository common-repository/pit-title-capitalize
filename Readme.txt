== Description ==
### PiT Title Capitalize by http://www.webingnet.com

This is a simple plugin. This plugin can convert your Blog title to Capitalize automatically. That means if you forget to Capitalize your Post/Blog title, its nothing if you have activated the PiT Title Capitalize Plugin in your WordPress powered Website.


== Installation ==

= From WordPress Plugins Directory =
1. Search 'Pit-Title-Capitalize' to Plugins library.
2. Install the plugin by click the Install option.
3. Activate the plugin.
4. That's all. Its working automatically.

= From your Desktop by FTP =
1. Download the 'Pit-Title-Capitalize.zip' file from WordPress plugins directory.
2. Upload to '/wp-content/plugins/' directory
3. Unzip the plugin folder
4. Activate the plugin through the 'Plugins' menu in WordPress Dashboard.
5. That's all. Its working automatically.
